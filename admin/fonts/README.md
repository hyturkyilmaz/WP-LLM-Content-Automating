# Fonts Directory

This directory is for optional TrueType/OpenType font files used by the GD image generator.

## Required Fonts (Optional)

The image generator will use these fonts if available:

- `Inter-Bold.ttf` — For headline text (recommended weight: 700)
- `Inter-Regular.ttf` — For footer/watermark text (recommended weight: 400)

## Download

Download Inter from Google Fonts:
https://fonts.google.com/specimen/Inter

## Fallback

If fonts are not present, the plugin automatically falls back to PHP's built-in GD fonts (larger, less refined text).
